Step 1: Initial Setup

Run the slash command to configure the system. This is a one-time admin-only step.

/setup-suggestions
  • Suggestion Channel: Select the channel (e.g., #suggestions) where messages will auto-convert to embeds.
  • Start Discussion: Toggle enabled/disabled. If enabled, adds a "Start Discussion" button to spawn threads for feedback.

Once set, LazaBot monitors the channel. No further commands required for users.
