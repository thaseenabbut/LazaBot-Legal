Feature Guide

Suggestion System

Turn any message into a clean, votable embed with live upvote/downvote buttons, real-time progress bars, and optional discussion threads. Setup in seconds.

Setup Guide
1
Initial Setup
One-time admin configuration — takes under a minute.
/setup-suggestions admin only
  • Suggestion Channel — Select the channel (e.g. #suggestions) where messages auto-convert to embeds.
  • Start Discussion — Toggle on to add a button that spawns a thread for feedback on each suggestion.
  • Once set, LazaBot monitors the channel. No further commands needed for users.
2
Submitting a Suggestion
No slash commands — just type in the channel.
"Add a #meme-channel for fun?"
  • LazaBot deletes the raw message to keep the channel clean.
  • Converts it into a polished embed with author credit and vote display.
  • Adds interactive 👍 Upvote and 👎 Downvote buttons automatically.
  • If Discussion is enabled, includes a button to spawn a dedicated thread.
3
Voting & Live Updates
The embed updates in real time on every vote.

  • Live Progress Bar — Green for approved, red for rejected. Updates instantly, no refresh needed.
  • Auto-Tags — Optional "🔥 Hot" tag triggers if approval exceeds 50% with 10+ votes.
4
Discussion & Resolution
Keep voting clean, discussion organized.
  • Click "Start Discussion" → instant thread spawns for replies, keeping the main channel focused on voting only.
  • Mods can review and resolve via buttons — no dedicated commands required.
  • Integrates with AutoMod to block toxic or duplicate submissions before they become embeds.
  • All actions are logged via webhooks for full audit trails. Anti-spam enforced by AccessNode pre-checks.
