1. Introduction

LazaBot is a Discord bot designed to assist Discord servers globally with moderation tasks, ensuring a seamless experience for diverse communities. This privacy policy explains how LazaBot collects, uses, and protects user data.

2. Data Collection

LazaBot collects:

  • Discord user IDs and usernames
  • Discord server IDs and names, channel IDs and names
  • Moderation action logs, including user IDs, action types, reasons, and snippets of message content if relevant
  • User preferences set by server administrators
  • Discord authentication information provided when logging into LazaBot’s dashboard

LazaBot does not collect emails, phone numbers, or full message content beyond moderation needs.

3. Purpose of Data Collection

Data is used to:

  • Perform moderation tasks like filtering messages, managing roles, and logging actions
  • Provide dashboard access for server administrators
  • Support server management and auditing

4. Data Storage and Retention

  • Data is stored on LazaBot’s servers, which are hosted on SantosHosting.
  • Discord authentication information from the dashboard is stored for 7 days.
  • When LazaBot is removed from a server, all associated data is deleted after 3 days, though this may not apply to backups or log data retained for auditing purposes.
  • Other data (e.g., moderation logs, preferences) is kept as long as necessary for service provision or auditing.

5. Security

We use reasonable security measures to protect data, though no system is 100% secure.

6. Access and Sharing

LazaBot developers access data for maintenance, improvement, and troubleshooting. Data is not shared with third parties except as required by law or with your consent.

7. Brand Features

By using LazaBot, you agree that we may use your server’s brand features (e.g., name and logo) for any reason, including but not limited to marketing purposes.

8. User Rights

You may request access, correction, or deletion of your data by contacting us at support discord server . Note that some data may remain in backups or logs after deletion requests.

9. Administrator Role

Server administrators configure LazaBot’s settings, affecting data collection. Users can address concerns through their server administrators, who may adjust settings or remove the bot.

10. Compliance

LazaBot complies with Discord’s terms Discord Developer Policy and applicable data protection laws.

11. Updates

We may update this policy. Changes will be posted here, and continued use of LazaBot indicates acceptance of the new terms.

12. Contact

If you have questions, concerns, or requests regarding this Privacy Policy or the management of your data, you can reach us through:
