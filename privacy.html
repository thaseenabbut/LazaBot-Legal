01

Introduction

LazaBot is a Discord bot designed to assist Discord servers globally with moderation tasks, ensuring a seamless experience for diverse communities. This Privacy Policy explains how LazaBot collects, uses, and protects your data.

02

Data Collection

LazaBot collects only what is necessary for its features to function. The following data may be collected:

Discord user IDs and usernames
Server IDs and names, channel IDs and names
Moderation action logs — including user IDs, action types, reasons, and snippets of message content where relevant
User preferences set by server administrators
Discord authentication information provided when logging into LazaBot's dashboard

LazaBot does not collect email addresses, phone numbers, or full message content beyond what is needed for moderation.

03

Purpose of Data Collection

Data is collected and used exclusively to:

  • Perform moderation tasks like filtering messages, managing roles, and logging actions.
  • Provide dashboard access for server administrators.
  • Support server management, auditing, and service improvement.
04

Data Storage and Retention

  • Data is stored on LazaBot's servers, hosted on SantosHosting.
  • Discord authentication information from the dashboard is stored for 7 days.
  • Other data (e.g., moderation logs, preferences) is kept as long as necessary for service provision or auditing.
  • Users may request manual deletion of their server's data at any time by contacting support.
30
Days to full deletion.
All server data — configurations, logs, and cache — is permanently deleted within 30 days of LazaBot's removal.

This ensures compliance with Discord's Developer Policy and industry-standard data privacy practices.

05

Security

We use reasonable security measures to protect all collected data. However, no system is completely secure, and we cannot guarantee absolute protection. We are committed to continuously improving our security practices.

06

Access and Sharing

LazaBot's developers may access data for maintenance, improvement, and troubleshooting purposes. Data is not shared with third parties except as required by law or with your explicit consent.

07

Brand Features

By using LazaBot, you agree that we may display your server's name and icon solely for showcasing bot usage or server count representation — for example, in website displays or promotional embeds. This is limited strictly to that purpose.

08

User Rights

You may request access, correction, or deletion of your data at any time by contacting us via our Support Discord Server. Requests are typically processed within 7–14 business days.

Submit a data request in LazaBot Support
09

Administrator Role

Server administrators configure LazaBot's settings, which directly affects what data is collected. Users can address privacy concerns through their server administrator, who may adjust settings or remove LazaBot from the server entirely.

10

Compliance

LazaBot complies with the Discord Developer Policy and applicable data protection laws. We are committed to keeping our practices current as regulations evolve.

11

Updates

We may update this Privacy Policy from time to time. Changes will be posted to this page, and continued use of LazaBot after an update indicates your acceptance of the revised policy. We recommend checking back periodically.

12

Contact

If you have questions, concerns, or data requests regarding this Privacy Policy, please reach out through our support server.
