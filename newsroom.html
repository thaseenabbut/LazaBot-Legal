February 26, 2026
LU47.2 Released: FlowControl Discontinued + Bug Fixes

In our ongoing commitment to stability and performance during the hiatus, we've rolled out LU47.2. This update focuses on streamlining the bot by removing underutilized features and addressing key bugs.

  • FlowControl Discontinued: Originally introduced in LU29 (February 16, 2025) for advanced command flow and conditional logic, FlowControl has been fully removed. It was causing occasional instability and had low user adoption. This change reduces overhead, making LazaBot leaner and more reliable without impacting core moderation tools.
  • Bug Fixes:
    • Fixed a rare crash in /warn command related to permission hierarchies (building on LU47.1).
    • Improved AFK status auto-removal: Now reliably clears when a user sends any message, preventing false positives.
    • Minor webhook logging tweaks for better event tracking in high-traffic servers.
    • General performance optimizations for faster response times on Event-Based AutoRoles and Advanced Suggestion System.

No new features added—focus remains on crash-proof reliability via AccessNode. All changes are live; restart your bot instance if needed. Join the support server for questions: discord.gg/pZbC5zJSCN.
